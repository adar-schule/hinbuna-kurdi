Dark mode, [YOUR CHOSEN COLORS].

Layout:
1. Top bar: Progress bar (3/8), X close button
2. Question: "What does 'spas' mean?"

   - Speaker icon
   - "Spas" text large

   - "Thank you" (correct answer)

   - "Goodbye"
